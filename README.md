# yo-mama-cooked : Spotify Playlist Generator

Turn your favorite music genres into a tasty playlist “dish”! Cooking Mama uses the Spotify Web API to blend together genres you choose and serve up a fresh, custom playlist—presented in a fun, recipe-style UI.

---

## Features

- **Genre “Ingredients”**  
  Let the user pick 2–5 music genres (e.g. “jazz,” “synthwave,” “lo-fi hip-hop”).
- **Automatic Playlist “Baking”**  
  Fetches tracks via the Spotify API and mixes them into a new, sharable playlist.
- **Recipe-Style UI**  
  Presents the creation process as an easy-to-follow recipe: ingredients list, step-by-step instructions, and a “serve” button to listen.
